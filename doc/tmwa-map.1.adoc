== Name

tmwa-map - The Mana World Athena map server

== Synopsis

*tmwa-map* [_--help_] [_--version_] [_--write-atcommand-config_ _outfile_] [_config-file_...]

== Description

*tmwa-map* is the map server of TMWA (The Mana World Athena), an
eAthena-derived MMORPG server.  It hosts the game maps, NPCs, monsters and
the scripting engine, and connects to the char server (and through it the
login server) to form a complete server.

The map server compiles and runs the TMWA scripting language; see
*tmwa-script*(7) for the language and its builtin functions.

This page is a brief stub.  Operational documentation (deployment, the
char/login/map split, world data layout) is not yet covered here.

== Options

_--help_::
Print a usage summary and exit.

_--version_::
Print the server version and exit.

_--write-atcommand-config_ _outfile_::
Write the default atcommand (GM command) configuration to _outfile_ and
exit.

_config-file_::
Path to a map-server configuration file.  Multiple files may be given and
are loaded in order.  When no configuration file is given, the server
loads `conf/tmwa-map.conf` from the working directory.

== Files

`conf/tmwa-map.conf`::
Default map-server configuration file, relative to the working directory.
